## Microservices Architecture

This project demonstrates a service based architecture using independent microservices
that communicate via HTTP requests.

### Services Included
- Inventory Service
- Pricing Service
- Order Processing Service

### Communication Pattern
- REST APIs
- JSON payloads
- Stateless services

### Why Microservices
- Separation of concerns
- Independent deployment
- Scalability

### Example Workflow
1. Frontend requests product pricing
2. Pricing service responds
3. Inventory service verifies availability
